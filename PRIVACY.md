# Privacy Policy

This plugin sends user prompts, conversation context, model parameters, tool definitions, and any request data supplied by Dify to the configured Skypool API endpoint so the selected model can generate a response.

The plugin stores only provider credentials inside Dify's credential storage. It does not persist prompts, model outputs, files, or API responses in the plugin runtime.

Skypool may process request metadata, usage records, billing records, and operational logs according to the Skypool service policies. Do not send secrets, personal data, or regulated data unless your Skypool account and deployment are approved for that use.

Users can remove the Skypool API key from Dify at any time by deleting the provider credentials or uninstalling the plugin.
